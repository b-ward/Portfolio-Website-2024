   - `#playGame` img: `style={{position:'absolute', width:'126px', height:'182px', left:'300px', top:'200px'}}`
   - `#instructions` img: `style={{position:'absolute', width:'126px', height:'182px', left:'650px', top:'200px'}}`

7. **Bootstrap Button replacement:**
   - `<Button className="made-button" ... onClick={...}>Creation</Button>` → `<button onClick={() => setMadeModalShow(true)} className="bg-accent text-black font-semibold px-4 py-2 rounded">Creation</button>`

8. **Modals:** Replace HelpModal and MadeModal inner Bootstrap modal patterns with shared Modal:
   ```jsx
   {helpModalShow && (
     <Modal title="Help" onClose={() => setHelpModalShow(false)}>
       {/* preserve exact help modal body content from original */}

   {madeModalShow && (
     <Modal title="How this was made" onClose={() => setMadeModalShow(false)}>
       {/* preserve exact made modal body content from original */}
     </Modal>

   ```
   Preserve all modal body content (text, images, links) verbatim.

10. **window.location.href navigation:** Leave unchanged (content preservation).
